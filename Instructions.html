STEPS ON HOW TO SET UP GIT, GITHUB, AND VISUAL STUDIO CODE

Git

Follow the steps indicated to set up Git on your Windows computer:

  1. Launch your browser and visit the official Git website at https://git-scm.com.
  2. Navigate to the Download section and choose your computer’s operating system.
  3. Click “Click here to download” to get the latest version of Git (2.51.2) for x64.
  4. After the download finishes, run the installer to complete the installation.
  5. To verify that Git has been successfully installed, right-click on your homescreen, and check if Git Bash appears in the menu. If it does, the installation was successful.

Visual Studio Code

Follow the steps indicated to set up Visual Studio Code on your Windows computer:

  1. Open your browser and visit the official Visual Studio Code website at https://code.visualstudio.com.
  2. Go to the Download section and select the operating system you are using.
  3. Click “Click here to download” to get the latest version of VS Code.
  4. After the download is complete, run the installer to set up VS Code on your computer.
  5. To verify the installation, open File Explorer, type cmd in the address bar, and press Enter.
  6. In the Command Prompt, type code and press Enter, this should launch Visual Studio Code.
  7. Once VS Code opens, you can customize the settings to your preference (e.g., auto-save, spacing, themes, and more).

GitHub Repository

Follow the steps indicated to set up Visual Studio Code on your Windows computer:

  1. Open your browser and visit the official GitHub website at https://github.com.
  2. Log in to your existing account, or sign up if you don’t have one yet.
  3. After logging in, click the “+” icon at the top-right corner of the page and choose “New repository.”
  4. Enter your repository name and an optional description, then set the visibility to Public.
  5. Click “Create repository” to complete the setup of your new GitHub repository.

Connecting Git to GitHub repository including essential commands

Take the following steps to connect you Git to your created GitHub repository:

  1. In VS Code, open Git Bash in the folder where you want to connect to your GitHub repository.
  2. Open your browser and go to https://codemy.com/git/ and copy the commands to Initialize Git.
    Enter these one by one to your Git bash terminal
    $ git init
    $ git config --global user.name "Your Name"
    $ git config --global user.email "you@youraddress.com"
    $ git config --global push.default matching
    $ git config --global alias.co checkout
  3. Next, copy the commands to connect to your GitHub repository from the same page.
    Enter these one by one to your Git bash terminal
    $ git init
    $ git add README.md
    $ git commit -m "first commit"
    $ git branch -M main
    $ git remote add origin (your repository URL)
    $ git push -u origin main
  4. Your local Git is now connected to your GitHub repository.
