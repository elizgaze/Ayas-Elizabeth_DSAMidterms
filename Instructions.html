Step-by-step instructions on how to set up Git, GitHub and VSCode

GIT SET-UP

Take the following steps to set up Git on your computer:

  1. Open your browser and go to the official website of Git: https://git-scm.com
  2. Go to download, then Select which type of operating system(os) you are using.
  3. Click the "click here to download" to download the latest vesion of Git(2.51.2)x64
  4. Once downloaded, click install
  5. To check if Git is now installed, go to an empty folder then right-click. If you see the Git bash, then it is installed.
